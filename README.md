### Running Locally

1. Open the project folder in VS Code
2. Install the "Live Server" extension (if you don't have it)
3. Right-click on `index.html` and select "Open with Live Server"
4. Allow camera access when prompted
5. Point your index finger at the camera and start creating!

### Controls

- **Clear Canvas**: Removes all drawn content
- **Toggle Trail Effect**: Switch between fading trails and solid lines
- **Color Picker**: Choose from neon cyan, magenta, yellow, green, or rainbow mode
